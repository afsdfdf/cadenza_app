# Cadenza iOS App Prototype

This workspace contains a mobile-first Apple-inspired prototype for the Cadenza music RWA project.

## Structure

- `archive/cadenza-site/`: mirrored public site and whitepaper
- `knowledge-base/`: project analysis and structured KB data
- `app/`: Vite frontend for the mobile app prototype

## App Highlights

- Apple-style mobile shell with bottom tab navigation
- Discover, Vault, Mint, Market, and DAO screens
- Simulated wallet connection
- Music RWA deal sheets and purchase actions
- Royalty vault and payout interactions
- Artist mint workflow for copyright NFT and fractional launch
- Governance voting flows

## Run

```bash
cd app
npm install
npm run dev
```

## Build

```bash
cd app
npm run build
```
